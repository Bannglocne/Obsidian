> The value of everything a seller must give up to produce a good


| Seller | Cost |
| ------ | ---- |
| A      | $900 |
| B      | 800  |
| C      | 600  |
| D      | 500  | 

> Because D is willing to do the work for $500 but gets $600 for doing it, we say that D receives [[Producer Surplus]] of $100.
