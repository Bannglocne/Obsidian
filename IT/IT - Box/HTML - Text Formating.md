# Text Formating
```html
<b></b> - Bold text
<strong></strong> - Important text
<i></i> - Italic text
<em></em> - Emphasized text
<mark></mark> - Marked text
<small></small> - Smaller text
<del></del> - Deleted text
<ins></ins> - Inserted text
<sub></sub> - Subscript text
<sup></sup> - Superscript text
```

# Quotation & Citation
```html
<blockquote></blockquote> - Quotations - Indent
<q></q> - Short Quotations - Insert ""
<abbr title=""></abbr> - Abbreviations - Short information when mouse over element
<address></address> - Contact Information - Italic and line breaks before and affter elemnt
<cite></cite> - Title of Creative Work - Italic
<bdo dir="rtl"></bdo> - Bi-Directional Override - Display right-to-left
```
