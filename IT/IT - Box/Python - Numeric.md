# Intenger
- Cú pháp: `<giá trị trả về> = int(<giá trị>)`
- Chỉ chuyển được số thực hoặc xâu của số nguyên
```python
>>> int("23")
23
```

# Float
- Cú pháp: `<giá trị trả về> = float(<giá trị>)`
- Chỉ chuyển được biểu thức số bất kỳ, xâu của số thực hoặc số nguyên
```python
>>> float(5)
>>> 5.0
```
