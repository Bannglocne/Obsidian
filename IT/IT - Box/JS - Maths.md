# Properties

```js
Math.E        // Euler's number  
Math.PI       // PI  
Math.SQRT2    // square root of 2  
Math.SQRT1_2  // square root of 1/2  
Math.LN2      // natural logarithm of 2  
Math.LN10     // natural logarithm of 10  
Math.LOG2E    // base 2 logarithm of E  
Math.LOG10E   // base 10 logarithm of E
```

| Methods                                    | Description                         |
| ------------------------------------------ | ----------------------------------- |
| Math.round(x)                              | rounded to nearest integer          |
| Math.ceil(x)                               | rounded up to nearest integer       |
| Math.floor(x)                              | rounded down to nearest integer     |
| Math.trunc(x)                              | the integer part                    |
| Math.sign(x)                               | if x is negative, null or positive. |
| Math.pow(x, y)                             | to the power of                     |
| Math.sqrt(x)                               | square root                         |
| Math.abs(x)                                | absolute value                      |
| Math.sin(x) / Math.cos(x)                  | given in radians                    |
| Math.min() / Math.max()                    | in a list of arguments              |
| Math.random()                              | between 0 and 1                     |
| Math.log(x) / Math.log2(x) / Math.log10(x) | base n logarithm of                 |
