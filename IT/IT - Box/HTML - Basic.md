# Meta
Character set used:
```html
<meta charset="UTF-8">
```
Keywords for search engines:
```html
<meta name="keywords" content="HTML, CSS, JavaScript">
```
Description of  web page:
```html
<meta name="description" content="Free Web tutorials">
```
Author of a page:
```html
<meta name="author" content="John Doe">
```
Refresh document every 30 seconds:
```html
<meta http-equiv="refresh" content="30">
```
Setting the viewport:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

# Layout
- `<header>` - `<footer>` : Header & Footer
- `<nav>` : Navigation links
- `<section>`  : Section
- `<article>` : Independent, self-contained content
- `<aside>` : Content aside from the content (Sidebar)
- `<details>` : Additional details that the user can open and close on demand
- `<summary>` : Heading for the `<details>` element

# URL
```
scheme://prefix.domain:port/path/filename
```
- **scheme** - **type** of Internet service (most common is **http or https**)
- **prefix** -a domain **prefix** (default for http is **www**)
- **domain** - Internet **domain name** (like w3schools.com)
- **port** - **port number** at the host (default for http is **80**)
- **path** - **path** at the server (If omitted: the root directory of the site)
- **filename** - name of a document or resource
