# Video
```html
<video width="50" height="50" controls autoplay muted>  
	<source src="movie.mp4" type="video/mp4">  
</video>
```

# Audio
```html
<audio controls autoplay muted>  
  <source src="music.mp3" type="audio/ogg">  
</audio>
```

# Plug-ins
```html
<object width="100%" height="100px" data="snippet.html"></object>
<embed width="100%" height="100px" src="snippet.html">
```

<iframe width="420" height="315"
