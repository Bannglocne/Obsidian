> **Exponential [[function]] with base $a$** $f(x) = a^x = \lim_{r \to x}a^r$

---
**Laws of Exponents**
  If $a > 0$ and $a \neq 1$, then $f(x) = a^x$ is a continuous [[function]] with domain $\mathbb{R}$ and range $(0, \infty)$. In particular, $a^x > 0$ for all $x$. If $0 < a < 1$, $f(x) = a^x$ is a decreasing [[function]]; if $a > 1$, $f$ is an increasing [[function]]. If $a, b > 0$ and $x, y \in \mathbb{R}$, then
 1. $a^{x+y} = a^x a^y$
2. $a^{x-y} = \frac{a^x}{a^y}$
3. $(a^x)^y = a^{xy}$
4. $(ab)^x = a^x b^

---
If $a > 1$, then
$$\lim_{x \to \infty} a^x = \infty \quad \text{and} \quad \lim_{x \to -\infty} a^x = 0$$

If $0 < a < 1$, then
$$\lim_{x \to \infty} a^x = 0 \quad \text{and} \quad \lim_{x \to -\infty} a^x = \infty$$

---
**[[derivatives]]**
$$
\frac{d}{dx}(a^x) = a^x \ln a
$$
---
**[[integral]]**
$$
\int a^x dx = \frac{a^x}{\ln a} + C \quad a \neq 1
$$
