#calculus #maths #derivatives 
Let $f$ be a [[function]] that satisfies the following hypotheses:
1. $f$ is [[continuity|continuous]] on the closed interval $[a, b]$
2. $f$ is [[derivatives|differentiable]] of the open interval $(a, b)$
Then there is a number $c$ in $(a, b)$ such that: $$f'(c) = \frac{f(b) - f(a)}{b - a}$$
or equivalently, $$f(b) - f(a) = f'(c)(b - a)$$

=> **Theorem**: If $f'(x) = 0$ for all $x$ in an interval $(a, b)$, the $f$ is constant on $(a, b)$

=> **Corollary**: If $f'(x) = g'(x)$ for all $x$ in an interval $(a, b)$, the $f - g$ is constant on $(a, b)$; that is, $f(x) = g(x) + c$ where $c$ is a constant
