# horizontal
The line $y = L$ is called a **horizontal asymptote** of the curve $y = f(x)$ if either $$\lim_{x \to \infty}f(x) = L \text{ or } \lim_{x \to -\infty}f(x) = L$$

# vertical
The line $x = a$ is a vertical asymptote if at least one of the following statements is true:
$\lim_{x \to a^{+}} f(x) = \infty$ - $\lim_{x \to a^{-}} f(x) = \infty$ - $\lim_{x \to a^{+}} f(x) = -\infty$ - $\lim_{x \to a^{-}} f(x) = -\infty$
