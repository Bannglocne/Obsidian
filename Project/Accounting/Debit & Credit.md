#accounting #economics 
> The term **debit** (Dr.) indicates the left side of an [[Account|account]].
> The term **credit** (Cr.) indicates the right side of an [[Account|account]].

We use the terms _debit_ and _credit_ in the recording process to describe where entries are made in [[Account|accounts]].


| **Comparing debit amounts and credits** |                |
| ----------------------------------- | -------------- |
| >                                   | Debit balance  |
| <                                   | Credit balance |
| ---                                 | ---            |
| **Change in cash**                      |                |
| Increase                            | Debits         |
| Decrease                            | Credits        |
